# 🚀 Customer Churn Prediction Project

Professional Machine Learning project to predict customer churn using real telecom customer data.

## 🔥 Key Results
- Accuracy: 79%
- ROC AUC Score: 0.83
- Best Model: XGBoost

## 📊 Features
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Random Forest
- XGBoost
- ROC Curve
- Feature Importance

## 🛠 Tools Used
Python, Pandas, Scikit-learn, XGBoost, Matplotlib, Seaborn
