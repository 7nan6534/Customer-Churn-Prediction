# 📊 Customer Churn Prediction using Machine Learning

## 📌 Project Overview

Customer churn is one of the biggest challenges for subscription-based businesses.  
This project uses Machine Learning techniques to predict whether a customer is likely to leave the company.

The goal is to help businesses identify at-risk customers early and improve retention strategies.

---

## 🚀 Project Objectives

- Predict customer churn accurately
- Compare multiple machine learning models
- Improve churn detection using XGBoost
- Visualize model performance
- Provide business insights for customer retention
